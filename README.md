# 📓 Folio - Notes Keeper App

A simple and clean notes keeper web app built with pure HTML, CSS, and JavaScript. No frameworks, no installation required.

## Features
- User login with password protection
- Create, edit, and delete notes
- Search notes by title or content
- Notes saved automatically in browser (localStorage)
- Fully responsive layout

## How to Run
1. Clone or download the repo
2. Open `index.html` in your browser
   — or use **Live Server** in VS Code for auto-reload
